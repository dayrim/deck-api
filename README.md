# Deck API

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# run db in docker
$ yarn run start:db
# run backend application
$ yarn run start
# go to https://localhost:3000/docs to see docs and try requests
```

## Testing the app

```bash
# setup testing db
$ yarn run test:migrate
# run e2e tests
$ yarn run test:e2e
```
